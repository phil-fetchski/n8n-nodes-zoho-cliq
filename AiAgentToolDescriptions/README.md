# Zoho Cliq n8n Node: AI Agent Tool Field Guides

This directory contains public, copy-ready guidance for configuring Zoho Cliq operations as AI tools in n8n.

Each operation guide includes:
- plain-language field descriptions you can paste into n8n field descriptions
- suggested `$fromAI()` expressions you can paste into field values
- suggested top-level Tool Description text you can paste after switching `Set Automatically` -> `Set Manually`
- required vs optional guidance for reliable agent behavior
