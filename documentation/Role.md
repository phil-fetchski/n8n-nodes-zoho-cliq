# Role Resource

The Role resource manages roles, role permissions, and user-to-role assignments in Zoho Cliq.

## Operations

| Operation | Description |
| --- | --- |
| Create Role | Create a role in the organization |
| Update Role | Update a role |
| Delete Role | Delete a role |
| Get Role | Get detailed information for a role |
| List Roles | Get roles in the organization |
| Add Users To Role | Add users to a role |
| Remove Users From Role | Remove users from a role |
| Get Users In Role | Get a list of users assigned to a role |
| Get Role Permissions | Get permission settings for a role |
| Add Role Permissions | Enable selected permissions for a role |
| Remove Role Permissions | Disable selected permissions for a role |
| Update Role Permissions | Update permission settings for a role |
| Build Permissions JSON Payload | Build permissions update JSON without making an API call |
