# Message Resource

The Message resource handles sending, editing, deleting, retrieving, and scheduling messages across channels, chats, bots, and users in Zoho Cliq.

## Operations

| Operation | Description |
| --- | --- |
| Post Message | Post a message to a channel, bot, chat, or user |
| Edit Message | Edit an existing message |
| Delete Message | Delete a message |
| Get Messages | Get messages from a chat, channel, or thread |
| Retrieve Message | Retrieve a specific message by ID |
| Schedule Message | Schedule a message to be sent later |
