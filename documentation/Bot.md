# Bot Resource

The Bot resource provides operations for interacting with bots in Zoho Cliq.

## Operations

| Operation | Description |
| --- | --- |
| Retrieve Bot Subscribers | Get list of users currently subscribed to a bot |
| Trigger Bot Calls | Trigger voice calls to users via bot |
