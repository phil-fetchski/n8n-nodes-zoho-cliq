# Database Resource

The Database resource manages Cliq Database storage records in Zoho Cliq.

## Operations

| Operation | Description |
| --- | --- |
| Create Record | Create a record in a Cliq Database storage |
| Update Record | Update a record in a Cliq Database storage |
| Delete Record | Delete a record from a Cliq Database storage |
| Get Record | Get a record by ID from a Cliq Database storage |
| List Records | List records in a Cliq Database storage |
