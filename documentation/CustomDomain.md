# Custom Domain Resource

The Custom Domain resource manages custom domain configuration for the Zoho Cliq organization.

## Operations

| Operation | Description |
| --- | --- |
| Add | Add custom domain settings |
| Verify | Verify custom domain and set status (active or inactive) |
| Get | Get custom domain settings |
| Delete | Delete the current custom domain settings |
