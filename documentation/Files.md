# File Resource

The File resource handles file retrieval and sharing across conversations in Zoho Cliq.

## Operations

| Operation | Description |
| --- | --- |
| Get File | Get a file using its file ID |
| Share Files | Share one or more files to channel, chat, bot, or user targets |
