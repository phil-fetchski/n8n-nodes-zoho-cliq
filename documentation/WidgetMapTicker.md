# Widget Map Ticker Resource

The Widget Map Ticker resource manages ticker entries within widget maps in Zoho Cliq.

## Operations

| Operation | Description |
| --- | --- |
| Add or Update Ticker | Add a new ticker or update an existing ticker in a widget map |
| Delete Ticker | Delete one or more existing tickers from a widget map |
