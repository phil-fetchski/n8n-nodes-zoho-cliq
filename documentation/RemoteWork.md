# Remote Work Resource

The Remote Work resource manages remote work status and availability in Zoho Cliq.

## Operations

| Operation | Description |
| --- | --- |
| Check In | Check in for remote work |
| Check Out | Check out from remote work |
| Get Remote Work Status | Get your remote work status and availability details |
