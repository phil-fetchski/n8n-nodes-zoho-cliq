# Chat Resource

The Chat resource manages direct message and group chat conversations in Zoho Cliq, including listing, muting, and pinned message management.

## Operations

| Operation | Description |
| --- | --- |
| List Chats | Get a list of chats across multiple chat types (dm, bot, chat, entity_chat) with optional filters |
| Get Chat Members | Get list of all members in a chat |
| Mute Chat | Mute notifications for a chat |
| Unmute Chat | Unmute notifications for a chat |
| Pin Message | Pin an existing message as the chat pinned message |
| Unpin Message | Remove the current pinned message from a chat |
| Get Pinned Message | Get the currently pinned message in a chat |
| Leave Group Chat | Leave a group chat |
