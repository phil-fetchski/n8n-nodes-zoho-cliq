# Reaction Resource

The Reaction resource manages emoji reactions on messages in Zoho Cliq.

## Operations

| Operation | Description |
| --- | --- |
| Add Reaction | Add a reaction to a message |
| Get Reactions | Get all reactions for a message |
| Remove Reaction | Remove a reaction from a message |
