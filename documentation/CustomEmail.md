# Custom Email Resource

The Custom Email resource manages the organization-level custom email configuration used for account notification emails in Zoho Cliq.

## Operations

| Operation | Description |
| --- | --- |
| Verify Custom Email | Retrieve the organization-level custom email configuration and verification state |
| Update Mail Configuration | Update the organization-level custom sender name, email address, and CNAME status used for notification emails |
