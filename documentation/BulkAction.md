# Bulk Action Resource

The Bulk Action resource provides data export operations for compliance and backup purposes in Zoho Cliq.

## Operations

| Operation | Description |
| --- | --- |
| Export Conversations | Export organization conversations (chats and groups) as CSV |
| Export Channels | Export organization channels as CSV |
| Export Members in a Conversation | Export members from a specific conversation as CSV |
| Export Messages | Export transcript and messages for a specific conversation |
