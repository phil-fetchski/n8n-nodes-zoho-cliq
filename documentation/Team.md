# Team Resource

The Team resource manages teams and team membership within the Zoho Cliq organization.

## Operations

| Operation | Description |
| --- | --- |
| Create Team | Create a team in the organization |
| Update Team | Update a team |
| Delete Team | Delete a team |
| Get Team | Get detailed information for a team |
| List Teams | Get teams in the organization |
| Add Team Members | Add members to a team |
| Delete Team Members | Delete members from a team |
| Get Team Members | Get list of members in a team |
