# User Resource

The User resource manages user profiles and organization membership in Zoho Cliq.

## Operations

| Operation | Description |
| --- | --- |
| Create User | Create a user in the organization |
| Update User | Update profile details for a user |
| Get User | Get detailed information for a user |
| List Users | Get users in the organization |
| Get User Teams | Get teams for a specific user |
| List User Layouts | Get available user profile layouts |
