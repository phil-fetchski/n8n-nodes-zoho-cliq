# Reminders Resource

The Reminders resource provides full reminder lifecycle management in Zoho Cliq, including creation, assignment, snoozing, completion tracking, and batch operations.

## Operations

| Operation | Description |
| --- | --- |
| Create Reminder | Create a reminder |
| Update Reminder | Update a reminder |
| Delete Reminder | Delete a reminder |
| Delete Reminders (Batch) | Delete multiple reminders |
| Get Reminder | Get detailed information for a reminder |
| List Reminders | Get reminders |
| Mark Reminder Complete | Mark a reminder as complete |
| Mark Reminder Incomplete | Mark a reminder as incomplete |
| Assign Users | Assign users to a reminder |
| Remove Reminder Assignees | Remove assignees from a reminder |
| Remind Assignee | Trigger a reminder nudge for one assignee |
| Remind Assignees | Trigger reminder nudges for selected assignees |
| Snooze Reminder | Snooze a reminder |
| Dismiss Snoozed Reminder | Dismiss snooze for a reminder |
| Clear Completed Reminders | Delete completed reminders by category |
