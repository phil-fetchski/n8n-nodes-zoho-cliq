# User Field Resource

The User Field resource manages custom user profile field schema definitions in Zoho Cliq. These operations work with field configurations (metadata), not individual user profile values.

## Operations

| Operation | Description |
| --- | --- |
| Add a User Field | Create a user-field schema definition (profile field configuration) |
| Update User Field Details | Update a user-field schema definition (name, options, rules) |
| Delete a User Field | Delete a user-field schema definition by field ID |
| Retrieve a Particular User Field | Retrieve one user-field schema definition by field ID |
| Retrieve All User Fields | List all user-field schema definitions configured for user profiles |
