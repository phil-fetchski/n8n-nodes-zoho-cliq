# Thread Resource

The Thread resource manages threaded conversations within channels, including creating threads, posting replies, managing followers, and controlling thread state.

## Operations

| Operation | Description |
| --- | --- |
| Create Thread | Create a new thread from a message |
| Post to Thread | Post a message to an existing thread |
| Schedule Message | Schedule a message to be sent in a thread |
| List Threads for Channel | Get a list of threads for a channel |
| Get Main Message | Get the main message that started a thread |
| Add Followers | Add users as followers to a thread |
| Remove Followers | Remove users from following a thread |
| Get Followers | Get the list of users following a thread |
| Get Non Followers | Get the list of channel members not following a thread |
| Follow Thread | Follow a thread (current user) |
| Unfollow Thread | Unfollow a thread (current user) |
| Auto Follow | Enable or disable automatic thread following in a channel |
| Update State | Close or reopen a thread |
