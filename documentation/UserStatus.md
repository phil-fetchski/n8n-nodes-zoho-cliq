# User Status Resource

The User Status resource manages user presence and status information in Zoho Cliq, including reusable custom statuses and transient (ephemeral) statuses.

## Operations

| Operation | Description |
| --- | --- |
| Add a New Status | Create a reusable custom status |
| Add a Transient Status | Set a temporary status using code, message, and expiry timestamp |
| Delete Status | Delete a reusable status by status ID |
| Delete Transient Status | Delete your currently active transient (ephemeral) status |
| Retrieve Current Status | Get your current status |
| Retrieve a User's Status | Get current user status for another user by user ID |
| Retrieve All Statuses | List all reusable custom statuses for the authenticated user |
| Update Current Status | Set an existing reusable status as your current status |
