# Channel Resource

The Channel resource provides full channel lifecycle management in Zoho Cliq, including creation, membership, permissions, bots, and moderation.

## Operations

| Operation | Description |
| --- | --- |
| Create Channel | Create a new channel |
| Update Channel | Update an existing channel |
| Delete Channel | Delete a channel permanently |
| Get Channel | Get detailed information about a specific channel |
| List Channels | Get a list of all channels with optional filters |
| Add Channel Members | Add members to a channel |
| Remove Channel Members | Remove a member from a channel |
| Remove Single Channel Member | Remove one channel member by ID or email |
| Get Channel Members | Get list of all members in a channel |
| Change Channel Member Role | Change role of a specific member in a channel |
| Change Channel Permission | Update channel permission objects for admin, moderator, and member roles |
| Join Channel | Join a channel (current user) |
| Leave Channel | Leave a channel (current user) |
| Archive Channel | Archive a channel (can be unarchived later) |
| Unarchive Channel | Restore an archived channel |
| Approve Channel | Approve a pending organization-level channel (admin only) |
| Reject Channel | Reject a pending organization-level channel (admin only) |
| Add Bot to Channel | Associate a bot with a channel by unique names |
| Remove Bot From Channel | Remove a bot from a channel using Bot ID |
