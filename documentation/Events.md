# Events Resource

The Events resource manages calendar events in Zoho Cliq, including creation, updates, RSVP status, and attachments.

## Operations

| Operation | Description |
| --- | --- |
| Create Event | Create an event |
| Update Event | Edit an event |
| Delete Event | Delete an event |
| Get Event Details | Get details for a specific event |
| Get Events | Get events from calendars |
| Update Event Status | Update your RSVP status for an event |
| Get Event Calendars | Get calendars available for events |
| Upload Event Attachment | Upload an attachment for an event |
