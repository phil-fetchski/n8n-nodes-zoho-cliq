# Calls and Meetings Resource

The Calls and Meetings resource provides access to call recording history and participant details in Zoho Cliq.

## Operations

| Operation | Description |
| --- | --- |
| List Call Recordings | List media session history and recordings |
| Get Recording Participants and Details | Get participant details for a media session recording |
