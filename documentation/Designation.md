# Designation Resource

The Designation resource manages designations and designation membership within the Zoho Cliq organization.

## Operations

| Operation | Description |
| --- | --- |
| Create Designation | Create a designation in the organization |
| Update Designation | Update a designation |
| Delete Designation | Delete a designation |
| Get Designation | Get detailed information for a designation |
| List Designations | Get designations in the organization |
| Add Designation Members | Add one or more users to a designation |
| Remove Designation Members | Remove one or more users from a designation |
| Get Designation Members | Get list of members in a designation |
